# CGPA Calculator (C++)

A simple **CGPA** calculator built using C++ for students to compute their Cumulative Grade Point Average based on course grades and credits.[web:1][web:8]

## Features

- Input number of subjects/courses for a semester.
- Enter marks or grades and credit hours for each course.
- Automatically convert marks/grades to grade points (according to a defined scale).
- Calculate total credits, total grade points, GPA for the semester, and overall CGPA.
- Simple console-based interface suitable for beginners in C++.

## Tech Stack

- Language: **C++**
- Type: Console application
- Build tools: Any C++17-compatible compiler (G++/Clang/MSVC).
